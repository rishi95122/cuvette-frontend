# How to start the project

1. **Clone the repository:**

    ```bash
    git clone https://github.com/rishi95122/cuvette-frontend.git
    ```
    
2. **Navigate to the project directory:**

    ```bash
    cd cuvette-frontend
    ```
    
3. **Install dependencies:**

    ```bash
    npm install
    ```
